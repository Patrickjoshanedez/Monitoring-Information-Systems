# BukSU Mentoring System

BukSU Mentoring System is an internal platform for Bukidnon State University that connects mentors and mentees,
offering role-based dashboards, guided mentoring sessions, and secure resource sharing. The application is currently
in beta testing and is not yet open to the public.

## Key Features

- Session booking and attendance tracking between mentors and mentees.
- Secure material sharing via Google Drive integration.
- Progress, goals, and feedback tracking per mentoring relationship.
- Real-time notifications and chat to keep both parties aligned.
